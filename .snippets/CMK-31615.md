## Remove unnecessary Jenkins API queue call
<!--
type: bugfix
scope: all
affected: all
-->

When awaiting `build_id_from_queue_item` the `queue_item.id` is handed over and the `queue_item` based on that id is then requested.
With this change, the `queue_item` is handed over if known to prevent a second API request to get the `queue_item` for further processing
